# AugmentedNeuronTracer
Augmented Neuron Tracer (ANT), a progressive and controllable semi-automatic neuron tracing tool in AR environment using Microsoft Hololens2 platform.
## Getting started
1. Install Unity 2021.3.26f1c1.
2. Clone this repository to your computer.
3. Open the cloned project "AugmentedNeuronTracer" with 2021.3.26f1c1.
